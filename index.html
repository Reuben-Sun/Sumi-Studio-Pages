隐私政策

最后更新：2025 年 5 月 23 日

概述

Sumi Studio 是一款完全在本地运行的 macOS 图像处理软件。我们尊重并保护您的隐私。本应用不会收集、存储、传输或分享任何用户数据。

数据收集

Sumi Studio 不会收集任何个人信息或使用数据，包括但不限于：

  • 个人身份信息（姓名、邮箱、电话等）
  • 设备信息
  • 位置数据
  • 使用行为或分析数据
  • 您编辑的照片或图像内容

网络访问

Sumi Studio 不需要网络连接即可运行。应用不会向任何服务器发送数据，也不会进行任何形式的网络通信。

本地数据

您的所有图像文件和编辑参数均保存在本地设备上，完全由您自己控制。应用不会将这些数据上传至任何云端服务。

第三方服务

Sumi Studio 不包含任何第三方分析工具、广告 SDK 或追踪代码。

政策变更

如果未来本政策有任何更新，我们将在此页面发布修改后的版本。

联系方式

如果您对本隐私政策有任何疑问，请通过 GitHub 联系我们：Sumi Studio

Privacy Policy

Last updated: May 23, 2025

Overview

Sumi Studio is a macOS image editing application that runs entirely offline. We respect and protect your privacy. This application does not collect, store, transmit, or share any user data.

Data Collection

Sumi Studio does not collect any personal information or usage data, including but not limited to:

  • Personal identifiers (name, email, phone number, etc.)
  • Device information
  • Location data
  • Usage behavior or analytics
  • Your photos or image content

Network Access

Sumi Studio does not require an internet connection to function. The application does not send data to any server or perform any network communication.

Local Data

All your image files and editing parameters are stored locally on your device, entirely under your control. The application does not upload any data to cloud services.

Third-Party Services

Sumi Studio does not include any third-party analytics tools, advertising SDKs, or tracking code.

Changes to This Policy

If this policy is updated in the future, the revised version will be posted on this page.

Contact
